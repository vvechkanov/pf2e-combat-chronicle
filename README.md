# PF2e Combat Chronicle

Foundry VTT module for Pathfinder 2e that automatically logs combat encounters in structured JSON format.

## Features (planned)

- Automatic combat tracking: rounds, turns, initiative
- HP tracking with deltas
- Effect/condition snapshots per turn
- Attack rolls and damage parsing from PF2e chat messages
- JSON export to JournalEntry
- Human-readable combat journal

## Requirements

- Foundry VTT v13+
- PF2e system 7.11+
